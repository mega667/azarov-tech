# azarov.tech

Personal resume and portfolio website for Konstantin Azarov, Senior Backend / Platform Engineer.

Live site: https://azarov.tech

## Overview

A responsive portfolio website highlighting my experience in backend/platform engineering, ecommerce systems, API integrations, authentication, reliability, and selected projects.

## Tech Stack

- Next.js
- React
- TypeScript
- CSS / Tailwind
- Vercel

## Features

- Responsive resume and portfolio layout
- Experience section with mobile-friendly collapsible cards
- Resume PDF view/download
- Project highlights
- Contact and mailto links
- Custom domain deployment

## Running Locally

```bash
npm install
npm run dev
```

Open:

```txt
http://localhost:3000
```

## Build

```bash
npm run build
```

## Deployment

Deployed on Vercel with custom domain:

https://azarov.tech
